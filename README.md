# CryptoTax Micro

Simple, affordable crypto tax reporting for small traders.

## Target Market
- Traders with <500 transactions per year
- Portfolio value: $100-$5,000
- Need basic FIFO/LIFO tax calculations
- Want to file taxes correctly without paying $50-300/year

## Pricing
$19/year - Simple, transparent, affordable

## Features (MVP)
- CSV import from major exchanges
- FIFO/LIFO cost basis calculation
- IRS Form 8949 generation
- PDF export for tax filing
- Email delivery

## Tech Stack
- Backend: Python + FastAPI
- Frontend: Next.js + Tailwind CSS
- Hosting: Vercel (serverless)
- Payments: Stripe
- Email: Resend
